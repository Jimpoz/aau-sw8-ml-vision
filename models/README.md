This folder stores platform-native model artifacts per facility.

Expected structure (generated by `training/rexport.py`):

- `models/{facility_id}/yolo26_indoor.onnx`        (server-side inference)
- `models/{facility_id}/yolo26_indoor.mlpackage`  (iOS offline model)
- `models/{facility_id}/yolo26_indoor.tflite`     (Android offline model)
- `models/{facility_id}/metadata.json`           (class names + export info)

The FastAPI server in `serving/` can also serve these artifacts through:
- `GET /v1/models/{facility_id}` (list)
- `GET /v1/models/{facility_id}/{platform}` (download)
